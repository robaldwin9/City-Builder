A webpage utilizing threejs to make procedurally generated cities.

  • Num Buildings: the number of randomly generated buildings
  • Min height: minimum height a building may possess
  • Max height: maximum height a building may have
  • Building X spacing: the amount of units in x direction between each pair of buildings
  • Building Z spacing: the number of units each building row is set apart
  • CityWidth: how spread out the city is in the X direction 20 means city falls in between -20 and 20 x units
  • Images from: texturer.com
